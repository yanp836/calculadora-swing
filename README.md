 ## Calculadora em Java (Swing)

Uma calculadora gráfica desenvolvida em Java, utilizando Swing, com foco em organização de código, separação entre lógica e interface e funcionalidades semelhantes a uma calculadora real.
Este projeto foi criado com o objetivo de praticar orientação a objetos, eventos, interfaces gráficas e boas práticas de organização.

### Funcionalidades
- Soma, Subtração, Multiplicação, Divisão, Suporte a números decimais, Backspace(apagar), Limpar (Clear).

- Interface Gráfica com Swing

- Estrutura do Projeto

### O projeto é dividido em duas partes principais:

- Lógica (Calculadora.java)
Responsável apenas pelos cálculos matemáticos.
Mantém o último resultado.
Métodos para operações matemáticas.
Tratamento de erros (ex: divisão por zero).

- Interface (CalculadoraSwing.java)
Responsável pela interface gráfica e interação com o usuário.
Captura eventos dos botões
Envia valores para a classe Calculadora
Exibe os resultados no display
Essa separação facilita manutenção, testes e evolução do projeto.

### Tecnologias Utilizadas
Java,
Java Swing,
Programação Orientada a Objetos,
Eventos e Listeners.
